Generate a presentation for the following topic or report.

Workflow:
1. Confirm format (Beamer, Quarto reveal.js, Marp, or plain Markdown) — default Quarto
2. Design the narrative arc: Motivation → Background → Methods → Results → Discussion → Conclusion
3. Create one slide per minute of talk time (estimate from content scope)
4. Every slide title is a complete declarative claim
5. Add 2–4 sentence speaker notes to every slide
6. Use LaTeX dollar notation for all math: `$\beta_1 = 0.42$`, `$$y_i \sim \mathcal{N}(\mu_i, \sigma^2)$$`
7. Flag where figures or tables should be inserted
